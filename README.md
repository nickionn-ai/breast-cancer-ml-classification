# Breast Cancer ML Classification

This project focuses on building a machine learning model to classify breast cancer tumors as malignant or benign using the Breast Cancer Wisconsin dataset.

## Project Overview

The goal of this project is to explore medical data and apply machine learning techniques to perform binary classification. Two models were implemented and compared:

- Logistic Regression
- Random Forest

## Dataset

The dataset used is the Breast Cancer Wisconsin dataset, available through the scikit-learn library. It includes clinical features extracted from digitized images of breast masses.

## Workflow

The project follows a complete machine learning pipeline:

1. Data loading and preprocessing
2. Exploratory data analysis
3. Feature scaling
4. Model training
5. Model evaluation
6. Model comparison

## Results

Both models achieved high performance, with Logistic Regression slightly outperforming Random Forest in this case.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Conclusion

This project demonstrates how machine learning can be applied to medical datasets to support classification tasks and gain insights from data.
